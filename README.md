# WeatherScript

Простая командная утилита для получения текущей погоды через OpenWeatherMap API.

## Перед запуском
1. Клонирование репозитория
2. Перейдите в проект
3. Создайте виртуальное окружение
4. Активируйте виртуальное окружение
5. Установите зависимости
6. Скопируйте в текущую директорию .env-example как .env
7. Вставьте API ключ с сервиса "openweathermap"

## Использование
python weather.py <Название города>

## Примеры вывода
- 9.63 light rain
- Произошла ошибка: city not found
